## Getting Started

Add this to your package's pubspec.yaml file:
```yaml
dependencies:
  masamune_sns: ^0.1.0
```
You should then run `flutter packages upgrade`.

## Usage

```yaml
import 'package:masamune_sns/masamune_sns.dart';
```
